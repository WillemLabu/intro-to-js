What is JavaScript?

Mozilla's JS page

An interpreted, object-oriented scripting language commonly most commonly used in browsers.

It is a prototype-based, dynamic, and supports object-oriented, imperative, and functional programming styles.

the basics of JS

Basics

JavaScript is case-sensitive.

Code instructions are called statements.

Statements and expressions should be separated with semi-colons (;).

Comments


  // This is a single line comment

  /* This is a multi-line comment, that is sometimes
     also referred to as a comment-block.
  */

  /* You can't, however, /* nest comments */ SyntaxError */

Variables

Variables define 'identifiers' - the name of the variable.

A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($)

Declarations


  var myVariable;

  var $the_Answer = 42;

  _name = "Willem";

Declarations


  var myVariable; // local, and = `undefined`

  let $the_Answer = 42; // local + block, and = 42

  _name = "Willem"; // global, and = 'Willem'

Scope


  var $the_Answer = 42;
  
  if ($the_Answer == 42) {
    let movie = "Hitchhiker's Guide to the Galaxy";
  }

  console.log(movie); // ReferenceError

Basic Control

Conditional statements


  if (condition) {
    statement_1;
  } else {
    statement_2;
  }

falsey values

  • false
  • undefined
  • null

Anything else will parse as truthy

Switch statements


  switch (expression) {
    case label_1:
      statements_1
      break;
    case label_2:
      statements_2
      break;
    
    // ...
    
    default:
      statements_default;
      break;
  }

for loop


  var step;
  for (step = 0; step < 5; step++) {
    // Runs 5 times, with values of step 0 through 4.
    console.log('I\'ve taken ' + (step + 1) + ' steps!');
  }

while loop


  while (variable > limit) {
    statements
  }

Careful not to loop infinitely!

for...in loops


  for (variable in object) {
    statements
  }

Functions

MDN Guide

Structure

  • The name of the function.
  • A list of arguments to the function, enclosed in parentheses and separated by commas.
  • The JavaScript statements that define the function, enclosed in curly brackets, { }.

Warning!

There are a few predefined functions in JavaScript

Don't call your functions any of those names, but more than that, know what they are so you can use them.

Basics


  function square(number) {
    return number * number;
  }
  • Parameter
  • Operator
  • Return

Parameter value


  function square(number) {
    return number * number;
  }

  var $the_Answer = 42;

  square($the_Answer);

  console.log($the_Answer); // This is still 42!

Anonymous Functions


  var square = function (number) {
    return number * number;
  }

What is Git?

It's a Version Control System.

Never lose work again! Save every file, and every step along the way, of your entire project.

It allows you to work with other people on the same code, without overwriting each other's work.

How does Git work?

There are three (conceptual) areas in any Git project:

  • Repository
  • Working Directory
  • Staging Area
